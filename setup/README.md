# Setup

Escribe aquí los pasos para ejecutar/probar tu proyecto y el nombre de los integrantes de tu equipo

1. Clone repository.
2. Go to the "recetario" directory.
```
cd recetario
```
3. Install dependencies
```
npm i
```
4. Run app
```
npx react-native run-android
```
